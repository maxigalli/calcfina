1. Instrucciones

Utilice esta calculadora para estimar la tasa de interés de un préstamo, a partir de los siguientes parmámetros:

Monto del Préstamo
Es la cantidad total prestada por el banco o institución financiera
Duración
Es el número de años otorgados para pagar el préstamo
Letra Mensual
Es el monto abonado mensualmente al préstamo
